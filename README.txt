# Tech Stack
- Jakarta EE 8
- JSF 2.3
- PrimeFaces 8.0

# Run the Application
- Clean and Build
- Deploy
- Clean and Build
- Run

# Additional Features
- Authenticaiton Filter: Only permit authenticated user to access the pages
- Custom Validation Checks: Uses Regex to check NRIC (ensuring starts with 'S' or 'T', 7 numerical characters and 1 alphabet), Mobile Phone (ensuring starts with '8' or '9', and a total of 8 digits)
- Responsive Navigation Bar & Pages: Adjusted according to the window size of the screen
- Return All Books/Pay total fine: Allows the users to return all books/pay total fine in one shot rather than returning the book/paying the fine one by one
